## Tugas 1 - Instalasi dan GIT

```bash
// 2311102100
// Verdi Pangestu

1. Pastikan aplikasi Git sudah ter-install di laptop atau komputer.
2. Siapkan folder penyimpanan bernama "Praktikum ABP" di dalam Local Disk C .
3. Jalankan CMD, kemudian arahkan path direktori menuju folder "Praktikum ABP" tersebut.
4. Unduh repository  Modul 1 dari GitHub menggunakan command gitclone.
5. Masuk ke dalam direktori repository dengan perintah  cd modul-1 .
6. Buat direktori pengumpulan tugas dengan  mkdir 231102100_Verdi Pangestu, lalu masuk ke dalamnya menggunakan cd 231102100_Verdi Pangestu.
7. Buat file laporan dengan menjalankan echo # Modul 1 > README.md .
8. Ketik  code . pada CMD agar folder langsung terbuka otomatis di VS Code .
9. Jika laporan dan tugas sudah selesai dikerjakan, kembali satu folder ke repository utama dengan  cd .. .
10. Siapkan file yang akan di-upload ke GitHub dengan menjalankan  git add . .
11. Berikan keterangan pada perubahan tugas menggunakan git commit -m "Tugas Modul 1" .
12. Sinkronkan data dengan server menggunakan git pull --rebase origin main .
13. Unggah file tugas ke branch main dengan perintah git push origin main .
14. Selesai.

```

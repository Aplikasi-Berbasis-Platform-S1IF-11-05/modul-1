# Dasar Teori

<p align="justify">
Git merupakan sistem <i>version control</i> terdistribusi yang digunakan untuk mengelola perubahan pada 
<i>source code</i> selama proses pengembangan perangkat lunak. Dengan Git, pengembang dapat melacak 
riwayat perubahan file, bekerja secara kolaboratif, serta menghindari konflik saat beberapa orang 
mengerjakan proyek yang sama. Platform seperti GitHub digunakan sebagai layanan penyimpanan 
<i>repository</i> berbasis cloud yang memungkinkan pengguna menyimpan, membagikan, dan mengelola 
proyek menggunakan Git melalui perintah <i>command line</i> seperti <b>git add</b>, <b>git commit</b>, 
<b>git pull</b>, dan <b>git push</b>.
